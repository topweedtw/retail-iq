# iPad Air (M4)

## 五大賣點（Selling Points）

> 📌 依 `AGENTS.md` §4.5 FAB+P 原則撰寫。Benefits 段落為 persona-adaptive，腳本生成器依所選客群挑選對應句。

### 1. M4 晶片 — 光線追蹤級效能，AI 任務飛速

- **F**：Apple M4 晶片（8 核心 CPU + 9 核心 GPU + 16 核心神經網路引擎） + **12GB 統一系統記憶體** + 硬體加速光線追蹤 [^1]
- **A**：較 M1 機型速度最快提升 2.3 倍；3D 專業算圖（光線追蹤）效能達 4 倍以上；神經網路引擎 AI 任務效能達 3 倍
- **B（學生）**：期末一次開八個 Safari 分頁 + Keynote + Goodnotes + Zoom，完全不掉速
- **B（創作者）**：DaVinci Resolve 4K ProRes 剪輯、Procreate Dreams 動畫、Octane X 3D 算圖，一鏡到底
- **B（商務）**：Apple Intelligence 即時幫忙總結 PDF、重寫郵件，效率拉滿

### 2. 兩種尺寸 — 11 吋便攜 vs 13 吋生產力

- **F**：11" (462g) 與 13" (616g) Liquid Retina 顯示器 [^1]
- **A**：iPad 系列中唯一提供「兩種尺寸且同規格」的機型
- **B（學生）**：11 吋書包隨身帶；13 吋宿舍桌面當筆電用
- **B（創作者）**：11 吋外出速寫手寫；13 吋繪圖創作空間大
- **B（家庭）**：11 吋小孩、長輩手拿剛好；13 吋看劇追片更過癮

### 3. Apple Pencil Pro — iPad 獨家互動體驗

- **F**：擠壓切換工具、桶式旋轉、雙指觸覺回饋、磁吸無線充電 [^1]
- **A**：其他觸控筆無此互動方式；是 iPad 與 Android 平板體驗最大差距
- **B（學生）**：上課手寫筆記，Smart Script 整理成正楷可貼文字
- **B（創作者）**：Procreate 畫畫像真筆一樣轉、捏、雙點切換工具
- **B（商務）**：PDF 文件簽名、白板會議記錄直接手寫標註

### 4. 端側 Apple Intelligence — 學習與效率加速

- **F**：書寫工具、Image Playground、Smart Script、Genmoji、視覺智慧 [^1]
- **A**：M3 神經引擎讓 AI 端側運算流暢；個資不離開 iPad
- **B（學生）**：報告用書寫工具改語氣、論文即時翻譯、手寫筆記變整齊
- **B（創作者）**：Image Playground 發想靈感、Smart Script 把草稿變清稿
- **B（商務）**：PDF 長文件一鍵摘要、英文合約即時翻譯

### 5. 一台裝置多場景 — 筆電／繪圖板／娛樂中心

- **F**：配 Magic Keyboard 變筆電、配 Pencil Pro 變繪圖板；支援 Stage Manager 多視窗 [^1]
- **A**：同一台硬體可秒切三種型態，市場上其他平板做不到
- **B（學生）**：上課筆電、下課繪圖、睡前追劇，一台搞定
- **B（創作者）**：工作室畫圖、外出拍攝隨身剪片、客戶會議當筆電
- **B（家庭）**：孩子用繪圖、爸媽用追劇、全家用視訊通話


## 三大獨家 Demo（Signature Demos）

> 📌 **展示原則**（依 `AGENTS.md` §4.2）：優先使用展示機預載素材，採「邊做邊說」節奏，口白每句 ≤ 15 字，先看效果再解釋原理。

### Demo 1. Apple Pencil Pro 擠壓手勢 + 桶式旋轉 — 從預載畫作開始

**為何獨特**：Pencil Pro 的「擠壓切換工具」與「桶式旋轉」是其他觸控筆沒有的互動方式；這是 iPad 系列與 Android 平板體驗拉開差距的關鍵。

**前置準備**：
- Demo 機預載 Procreate 或 Freeform，**已內建官方範例畫布／插畫樣本**
- Pencil Pro 配對且充飽電
- 打開其中一張範例畫作，畫布已在主畫面

**分步腳本**（邊做邊說）：
1. 👉 打開預載範例畫作
   🗣️ 「您看，這是用 iPad 畫的。」
   👀 客戶看到一幅已完成的插畫／手寫筆記
2. 👉 拿 Pencil 在畫布上補畫幾筆
   🗣️ 「我現在加幾筆 — 像真的筆。」
   👀 低延遲、壓感自然
3. 👉 擠壓筆桿 → 工具列彈出
   🗣️ 「一捏，工具就到。」
   👀 工具選單出現在筆尖旁
4. 👉 切到書法筆 → 旋轉筆桿畫一條線
   🗣️ 「像真筆一樣轉。」
   👀 筆觸粗細隨旋轉變化
5. 👉 故意畫錯 → 雙指點擊筆桿
   🗣️ 「雙點，撤銷。」
   👀 觸覺震動回饋 + 線條消失

**客戶親手做**：請客戶接手畫一個愛心或寫自己名字，全程只用擠壓／旋轉／雙點擊切換工具，不碰螢幕 UI。

**常見卡點與應變**：
- 若 Pencil 沒電 → 吸附到 iPad 側邊磁吸充電，順勢展示無線充電
- 若預載畫作被上個客戶刪掉 → 當場新建一張，先畫一個 Apple Logo 當「既有畫面」再往下示範
- 若客戶說「我不畫畫」→ 轉向 Notability／備忘錄 手寫筆記場景，接 Demo 3

**收尾金句**：「懂您手勢的筆 — 這是 iPad 的專利。」

### Demo 2. Stage Manager + Magic Keyboard — 從預開多視窗直接變身

**為何獨特**：iPad Air 配合 Magic Keyboard + Stage Manager 可做到真正的桌面級多工（最多 4 視窗同時 + 外接螢幕），而且隨時拆下就是平板 — 市場上沒有其他平板能如此無縫切換兩種型態。

**前置準備**：
- Demo 機已裝 Magic Keyboard for iPad Air
- **Stage Manager 預先開啟**，**已預開三個 app**：Safari、備忘錄、行事曆（層疊排好）
- 從 Magic Keyboard 先拆下 iPad，以「平板態」交給客戶

**分步腳本**（邊做邊說）：
1. 👉 單手把平板遞給客戶
   🗣️ 「先當平板看劇。」
   👀 客戶感受輕巧
2. 👉 客戶握著平板的同時，放回 Magic Keyboard
   🗣️ 「要工作？一秒變身。」
   👀 磁吸定位，鍵盤觸控板即時啟用
3. 👉 輕點螢幕，展示三個預開視窗層疊
   🗣️ 「三個 app 都開著了。」
   👀 客戶看到桌面級多工畫面
4. 👉 拖拉視窗重疊、從側邊切換最近 app
   🗣️ 「像 Mac 一樣切。」
   👀 流暢無卡頓
5. 👉 再次從 Magic Keyboard 上拎起 iPad
   🗣️ 「帶走平板，繼續用。」
   👀 三個 app 狀態完整保留

**客戶親手做**：請客戶自己把 iPad 裝上、拿下鍵盤 2-3 次，親手體驗磁吸力道與定位精準度。

**常見卡點與應變**：
- 若 Stage Manager 沒開啟 → 控制中心一鍵打開，同時說「這功能可隨時開關」
- 若預開的 app 不見 → 即場開 Safari + 備忘錄兩個，仍能示範核心切換
- 若客戶只想當平板用 → 強調可後期再加購鍵盤，不必現在決定

**收尾金句**：「一台 iPad，兩個身份。」

### Demo 3. Smart Script 手寫整理 — 從預載筆記樣本切入

**為何獨特**：iPadOS 的 Smart Script 支援繁體中文手寫筆跡美化與重排，能保留個人筆跡風格但整齊度大幅提升；這在 Android 平板生態系是空白地帶。

**前置準備**：
- 展示機備忘錄 App **預載一份官方手寫筆記樣本**（內容為會議／課程筆記，包含中英文）
- Apple Pencil Pro 配對就緒
- 打開該份預載筆記，畫面停在可編輯狀態

**分步腳本**（邊做邊說）：
1. 👉 打開預載手寫筆記樣本
   🗣️ 「您看這份手寫筆記。」
   👀 客戶看到有一點潦草但可讀的筆跡
2. 👉 選取一段文字 → 長按 → 選「讓筆跡更整齊」
   🗣️ 「按一下。」
   👀 字跡瞬間變整齊，仍保有個人風格
3. 👉 在中間插入三個字（例：「跟朋友」）
   🗣️ 「中間插字。」
   👀 iPad 自動把右邊文字往後推，排版保持整齊
4. 👉 圈選一段 → 複製 → 切到訊息 App 貼上
   🗣️ 「手寫直接變文字。」
   👀 客戶看到手寫變純文字訊息

**客戶親手做**：請客戶用 Pencil 自己寫一段待辦清單或心情日記（中文 OK），體驗 Smart Script 的即時整理。

**常見卡點與應變**：
- 若預載筆記被清掉 → 當場請店員先寫一段「範例」作為「客戶的筆記」再示範整理
- 若客戶字跡太潦草辨識率低 → 示範「畫圓選取 → 重排」的分步流程
- 若客戶從未用手寫筆 → 先用「隨便塗鴉 → 變方正形狀」的 Shape Recognition 暖身

**收尾金句**：「手寫的溫度，數位的整齊。」

## 常見客戶問題與回應

**Q1：iPad Air 跟 iPad Pro 差在哪？**
A：三大主要差異 — Pro 有 ProMotion 120Hz + OLED 顯示器、M4 晶片、更薄機身。Air 的 M3 對多數人已足夠。建議對比 [[comparisons/ipad-air-vs-pro]]。

**Q2：可以取代筆電嗎？**
A：看用途。文書、筆記、上網、影音、輕度創作完全可以。但若需要跑 Windows 專用軟體、Xcode、複雜多視窗作業，MacBook 還是更適合。

**Q3：11 吋還是 13 吋？**
A：常外出 / 當筆記本用 → 11 吋；常在家 / 繪圖 / 多工 → 13 吋。

**Q4：需要買鍵盤和 Pencil 嗎？**
A：看使用情境。寫報告 → 建議鍵盤；畫畫／手寫筆記 → 建議 Pencil。可以單買所需配件。

**Q5：128GB 夠用嗎？**
A：以影音娛樂、輕度使用足夠。若常離線下載影片或需存大量照片，建議 256GB 起。

## 常見反對意見處理

- **客戶說**：「我已經有 MacBook，還需要 iPad 嗎？」
  **你可以回應**：兩者互補。iPad 的優勢在手寫、觸控繪圖、輕便閱讀、Sidecar 當 Mac 第二螢幕。建議想想是否有這些使用情境。

- **客戶說**：「我用 iPad mini 就夠了」
  **你可以回應**：iPad mini 便攜性強，但螢幕較小不利多工與創作。Air 的 11 吋是便攜與生產力的平衡點。

- **客戶說**：「價格加上配件就跟 MacBook Air 差不多了」
  **你可以回應**：確實配齊後接近 MacBook Air 價格。差別在使用型態 — iPad 是觸控 + 筆 + 分離鍵盤；MacBook 是傳統筆電。可以從您主要的工作方式來選擇。
