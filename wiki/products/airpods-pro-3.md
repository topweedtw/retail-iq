## 五大賣點（Selling Points）

> 📌 依 `AGENTS.md` §4.5 FAB+P 原則撰寫。

### 1. 世上最強入耳 ANC — 降噪效果較 Pro 2 提升 2 倍

- **F**：較 AirPods Pro 2 提升最高 **2 倍**、較首代 Pro 提升 **4 倍**的主動式降噪效果（IEC 60268-24 測試） [^1]
- **A**：搭配升級的被動隔音封閉 + 全新超低噪音麥克風，結合運算音訊降低環境噪音
- **B（通勤族）**：捷運、公車、高鐵車廂噪音幾乎消失，不必把音量開很大
- **B（遠距工作）**：咖啡廳、共享空間、機場會議不再干擾思路
- **B（創作者）**：剪片、混音、寫作時的專注環境

### 2. IP57 防塵抗汗水 — 首款運動級 AirPods

- **F**：**IP57** 等級（首款 AirPods 達此標準）、耳塞套內旋設計更穩固 [^1]
- **A**：同時防塵（6）+ 抗汗水（7），適合各種非水上運動；加上泡棉微球隔層讓運動中更貼合
- **B（跑者）**：大汗淋漓 + 突然下雨，耳機安然無恙
- **B（健身房）**：重訓甩頭、HIIT、滾筒按摩不怕掉
- **B（戶外騎車）**：塵土、偶爾細雨都不怕

### 3. 心率感測 — 健身教練住進耳朵裡

- **F**：歷來最小巧的光學心率感測器（每秒 256 次 LED 脈衝 + 加速度計融合），支援 50+ 運動類型 [^1]
- **A**：AirPods 變成戴在耳上的心率帶 / Polar 裝置替代品，不必另戴 Apple Watch 也能追蹤
- **B（偶爾健身）**：想知道跑步心率但不想戴 Watch → AirPods 就行
- **B（瑜伽 / 太極）**：靜態運動中追蹤心率，衡量入定深度
- **B（慢活族）**：Fitness app 整合走路、健身車、HIIT 等數據

### 4. 即時翻譯 — 出國不用 Google 翻譯

- **F**：透過配對 iPhone 的 Apple Intelligence 進行即時翻譯，對方語言聽起來是你的語言；你回話用母語，iPhone 螢幕顯示對方語言 [^1]
- **A**：若雙方都戴 AirPods Pro 3 → 各自用母語對話，各自聽翻譯（支援特定語言 / 地區）
- **B（出差族）**：外國會議、客戶拜訪不必帶翻譯
- **B（旅行者）**：問路、點餐、購物自然溝通
- **B（跨文化家庭）**：與異國親屬溝通沒隔閡

### 5. 助聽器功能 — 醫療等級聽力輔助

- **F**：支援「聽力測試」+「助聽器模式」（18 歲以上、自覺輕度至中度聽損人士）+「降低高音量」+「對話增強」 [^1]
- **A**：首款把醫療等級聽力輔助整合進主流耳機的消費性電子產品（已通過美 FDA）
- **B（聽損人士）**：一副耳機取代昂貴的助聽器（不取代專業診斷）
- **B（年長父母）**：送長輩一副，兼具日常聽音樂 + 對話增強
- **B（樂手 / DJ）**：「降低高音量」保護聽力不受演出損害


## 三大獨家 Demo（Signature Demos）

> 📌 展示重點：**「聽得到」的體驗**最難用口白描述，必須讓客戶親耳試戴。

### Demo 1. ANC A/B 切換 — 店內噪音環境下的親耳對比

**為何獨特**：降噪效果必須「感受」才有說服力。A/B 切換法讓客戶自己體驗 Pro 3 vs 無降噪的差別。

**前置準備**：
- 店內應有持續背景噪音（音樂 + 人聲）
- Demo AirPods Pro 3 已消毒、配對 Demo iPhone
- 準備音樂列表（推薦有低音 + 人聲混合的曲目）
- 有 Demo 用的 Pro 2 耳機（若可能）做 A/B 比較

**分步腳本**：
1. 👉 請客戶戴上 Pro 3，播放音樂
   🗣️ 「先聽聽店內環境音。」
   👀 ANC 關閉 → 客戶能清楚聽到店內雜音
2. 👉 切換至 ANC 模式
   🗣️ 「切降噪，再聽一次。」
   👀 客戶感受噪音瞬間消失 → 常見表情：驚訝、抿嘴
3. 👉 切到通透模式
   🗣️ 「現在是通透模式，聽得到人聲。」
   👀 客戶清楚聽見店員問話
4. 👉 捏握梗部 → 切回 ANC
   🗣️ 「捏一下切換，很直覺。」
5. 👉 （可選）對比 Pro 2
   🗣️ 「這是 Pro 2 — 你自己感受差別。」

**客戶親手做**：讓客戶戴著 Pro 3 連續聽 2 分鐘，習慣寂靜感。

**常見卡點**：
- 客戶說「我聽不出差別」→ 把音量降到最低，讓 ANC 負擔全部噪音屏蔽工作
- 耳塞不合 → 換 XS / S / M / L / XXS 試試

**收尾金句**：「不必把音量開大，就能聽得清楚。」

### Demo 2. 心率追蹤 — 健身房場景模擬

**為何獨特**：AirPods 能取代 Apple Watch 做心率追蹤是破壞式功能；展示方式要讓客戶看到「AirPods = 健身器材」的概念。

**前置準備**：
- Demo iPhone 已裝 Fitness app + 連結 AirPods
- 店內有可以輕度運動的區域（至少能踏步 / 做深蹲）

**分步腳本**：
1. 👉 戴上 Pro 3 → 打開 Fitness app → 選「HIIT」
   🗣️ 「不用戴 Watch，心率也能追。」
2. 👉 iPhone 螢幕即時顯示心率數字
   🗣️ 「靜止時 72。」
3. 👉 請客戶做 10 下深蹲或原地高抬腿
   🗣️ 「動起來試試。」
   👀 心率從 72 上升至 100+
4. 👉 停下 → 心率曲線回落
   🗣️ 「停下來看回復。」
   👀 完整的運動 → 回復曲線
5. 👉 結束訓練 → 查看消耗卡路里 + 心率區間

**客戶親手做**：讓客戶親自踏步或跳動 30 秒，看 AirPods 能否精準抓心率。

**常見卡點**：
- 耳塞太鬆 → 心率訊號會雜訊 → 強調耳塞貼合度的重要
- 客戶有 Apple Watch → 強調「運動時只戴 AirPods 也行」的便利

**收尾金句**：「耳朵裡的健身教練。」

**為何獨特**：即時翻譯是 Apple Intelligence 實用落地的顯著案例；最能打動有出國或跨文化溝通需求的客戶。

   🗣️ 「你自然回話，對方看螢幕。」

**收尾金句**：「溝通不再因為語言而停下。」

## 常見客戶問題與回應

**Q1：Pro 3 跟 Pro 2 差在哪？值得升級嗎？**
A：五大升級 —（1）降噪效果 **×2**；（2）**IP57** 防塵抗水（Pro 2 只有 IPX4）；（3）新增**心率感測**；（4）電池 **8h**（Pro 2 是 6h）；（5）**5 種尺寸**耳塞（新增 XXS）。值得升級條件：常運動 + 常出國 + 聽損需求；若你 Pro 2 還沒壞且用得愉快，可以再戰一兩年。

**Q2：Pro 3 跟 AirPods 4 ANC 款差在哪？**
A：Pro 3 是 **入耳式**（更好的 ANC、更穩定運動、有心率感測）；AirPods 4 ANC 是 **非入耳式**（更舒服、沒心率、ANC 較弱）。想要極致降噪 + 運動 → Pro 3；不喜歡異物感 → 4。

**Q3：Pro 3 跟 AirPods Max 2 差在哪？**
A：Max 2 是**耳罩式**，音質最佳、電池最久、但重、大、不適合運動。Pro 3 攜帶方便、運動可用、有 IP57。音樂發燒友 → Max 2；日常通勤 + 運動 → Pro 3。

**Q4：助聽器功能真的能取代助聽器嗎？**
A：對於輕度至中度聽損的 18 歲以上人士，**可以作為輔助使用**（美 FDA 通過）。但不取代專業聽力師診斷，重度聽損或兒童仍需專業助聽器。

**Q5：電池壽命可以換嗎？**
A：AirPods 電池為消耗品，老化後可透過 AppleCare+ 或付費維修更換。一般正常使用 2-3 年電池會明顯退化。

## 常見反對意見處理

- **客戶說**：「我用 Sony WF-1000XM5，降噪也很好」
  **你可以回應**：Sony XM5 確實優秀，但 Pro 3 的優勢 —（1）與 iPhone 的**無縫切換**（電話 / iPad / Mac 自動切）；（2）**Apple Intelligence 即時翻譯**；（3）**心率感測**+IP57；（4）充電盒「精確尋找」距離 1.5×。若你全是 Apple 生態 → Pro 3 整合度贏。

- **客戶說**：「價格好貴」
  **你可以回應**：可以這樣算 — 取代 ANC 耳機（8000+）+ 運動心率帶（3000+）+ 基本助聽器（20000+）+ 即時翻譯裝置。分開買更貴。再加上蘋果生態的整合價值。

- **客戶說**：「入耳式不習慣，耳朵會悶」
  **你可以回應**：Pro 3 新的多孔聲學架構實際上**降低了入耳的壓迫感**；加上 5 種尺寸耳塞，一定能找到合適你的。建議試戴 10 分鐘再判斷。

- **客戶說**：「我已經有 Apple Watch 測心率了」
  **你可以回應**：Pro 3 的心率是「**不想戴 Watch 時的備案**」— 例如只想輕鬆散步、游泳前洗澡時、健身專注時不戴手錶。兩者其實互補。
