## 🎭 三種風格一覽

### Style 1 · Apple HIG 極簡

- **靈感**：apple.com、macOS Settings、HIG
- **調色**：#fbfbfd 背景、#1d1d1f 文字、#0071e3 強調
- **字型**：SF Pro stack
- **形狀**：大圓角 (18px+)、細邊框、微妙陰影
- **互動**：hover 微浮起、毛玻璃導覽列
- **氣質**：**沉穩、專業、熟悉**

**適合**：
- 員工一看就知道「這是自家系統」
- 客戶臨時看到螢幕也不覺得業餘
- 管理層審美要求高

**不適合**：
- 學習黏著度目標高（太冷靜）
- 需要強化「遊戲化刺激」

---

### Style 2 · Gamified 遊戲化

- **靈感**：Duolingo、Quizlet、Headspace
- **調色**：#faf7f2 暖底、綠橘紫粉漸層、#58cc02 成功色
- **形狀**：粗圓角 + 4px 按鈕陰影（按下會「陷下去」）
- **元件**：徽章、連續天數、XP、等級條、排行榜、情境對決
- **互動**：confetti 飄浮、按鈕彈跳
- **氣質**：**熱情、激勵、年輕**

**適合**：
- 門市年輕員工、遊戲化學習
- 新人訓練階段（前 3 個月）
- 希望員工「每天打開」的黏著設計

**不適合**：
- 商務嚴肅形象偏好
- 資深員工可能覺得幼稚
- 高齡使用者

---

### Style 3 · Pro Dashboard 專業資料密集型

- **靈感**：Linear、Raycast、Vercel Dashboard
- **調色**：#0b0d10 暗底、灰階為主、#7aa2ff 強調
- **字型**：SF Mono + SF Pro 混排
- **元件**：Sidebar 導覽、指令搜尋列（⌘K）、Tier 徽章、鍵盤快捷鍵
- **資訊密度**：極高（三欄同時顯示：輸入 / 預覽 / 來源）
- **氣質**：**效率、硬核、稽核友善**

**適合**：
- 管理員後台（看 ingest status、lint 報告）
- 資深店長需要快速操作
- 合規稽核（來源追溯、Tier 標記一目了然）

**不適合**：
- 一般門市人員（門檻高）
- 偏重觸控的 iPad 現場（太 desktop-oriented）

## 🧭 如何選擇？

**建議採用混合策略**：

```
┌─────────────────────────────────────────┐
│  Front-end for 門市店員（iPad 現場）   │
│  → Style 1 (Apple HIG)                   │
│  → 或 Style 2（若團隊年輕、學習導向）  │
└─────────────────────────────────────────┘
                   ↕
┌─────────────────────────────────────────┐
│  管理後台（桌機為主）                    │
│  → Style 3 (Pro Dashboard)               │
└─────────────────────────────────────────┘
```

| 決策題 | 選 Style 1 if... | 選 Style 2 if... | 選 Style 3 if... |
|---|---|---|---|
| 主要使用者 | 資深店員 + 管理層 | 新進年輕店員 | 管理員、系統維運 |
| 使用場景 | 門市現場、展示桌 | 休息室學習時段 | 辦公室後台 |
| 品牌協調 | 與 apple.com 一致 | 強化「學習感」 | 效率工具感 |
| 情感氛圍 | 專業信任 | 活潑激勵 | 高效掌控 |

## 🚀 下一步建議

**週一 Review 後**：

1. **選定主方向**（或決定雙軌）
2. **把選定風格轉為 Next.js 元件**（拆成 shadcn/ui 可重用元件）
3. **設計系統抽取**：把顏色、spacing、字型抽成 design tokens
4. **做 2-3 個關鍵 flow 的高保真版本**：
   - 腳本生成器完整流程（最重要）
   - 測驗挑戰 run-through
   - 週報閱讀體驗

## 💡 額外觀察

設計過程中發現的幾個**不分風格都應該做**的事：

1. **P 層填空的 UI 要特別照顧**：輸入時應該有提示詞（「想想您自己的真實經驗...」）
2. **合規狀態必須視覺化**：即使 Style 1 沒呈現，正式版仍要有
3. **離線狀態提示**：門市 iPad 可能 Wi-Fi 不穩，必須處理 offline 模式
4. **可匯出**：腳本必須能匯出 PDF / 列印卡片格式
5. **個人腳本庫**：每位店員的 P 層歷史版本要能回看
