# Apple Watch Ultra 3

## 一句話定位

**極致的運動探險錶**。49mm 鈦金屬錶殼、**100 公尺防水（可支援 40 公尺休閒水肺潛水）**、**42 小時電池**、雙頻 GPS 與 Action Button，為登山、三項鐵人、自由潛水而生。[^1]

## 五大賣點（Selling Points）

> 📌 依 `AGENTS.md` §4.5 FAB+P 原則撰寫。

### 1. 42 小時電池 — 整場鐵人三項 / 多日登山一顆搞定

- **F**：正常使用 **42 小時**、低耗電模式 **72 小時**（Series 11 是 24h / 38h） [^1]
- **A**：支援普通使用者一連三日不充電；或支援 IRONMAN 226 全程 + 隔天回顧
- **B（鐵人三項）**：游泳 2.4mi + 騎車 112mi + 跑步 26.2mi，一顆電池完成，含賽後 24 小時恢復數據
- **B（登山者）**：3 天 2 夜百岳縱走不帶行動電源，睡眠追蹤 + GPS 全程開
- **B（超馬選手）**：100 公里超馬 + 途中 8 小時睡眠補給，仍有電記錄衝線

### 2. 100m 防水 + Oceanic+ 潛水 — 從游泳池到 40m 潛水一錶到底

- **F**：ISO 22810 **100 公尺**防水、搭配 Oceanic+ app 支援 **40 公尺休閒水肺潛水** [^1]
- **A**：同時涵蓋高速水上運動（滑水、帆船）+ 休閒潛水（小於專業深潛的深度）
- **B（水肺潛水者）**：下水前輸入潛水計畫、途中監控無減壓限制、出水後自動產生潛水 log
- **B（衝浪 / 帆船）**：能監控海浪、風速、GPS 軌跡，不必額外帶潛水錶
- **B（泳訓者）**：自動追蹤泳姿、距離、SWOLF、每趟速度

### 3. Action Button — 一鍵觸發常用任務

- **F**：機身左側橘色 **Action Button**，可客製化 9 種快速動作（訓練 / 路線標記 / 對講機 / 手電筒 / 潛水 / 新增航點 / 翻譯 / 快捷指令…） [^1]
- **A**：Ultra 系列獨有；讓你戴手套、濕手時也能一鍵啟動
- **B（跑者）**：比賽起跑前按一下開始計時，不必左滑右點
- **B（登山者）**：按一下標記航點，回程時自動導航
- **B（潛水者）**：下水按一下切 Oceanic+，出水按一下停止

### 4. 雙頻 GPS — 業界最強定位精準度

- **F**：雙頻 GPS（L1 + L5），2025 年 8 月測試中較當時市面領先雙頻運動錶更準 [^1]
- **A**：在都市高樓峽谷、密林、深谷等訊號挑戰場景仍能精準描繪路徑
- **B（都市跑者）**：台北信義區、紐約曼哈頓高樓群繞圈，路徑仍精準
- **B（越野跑）**：阿爾卑斯、北插天山密林，GPS 不漂移
- **B（軍警）**：山區搜救、野外訓練，定位信心滿滿

### 5. 警報器 + 回溯 — 戶外安全雙保險

- **F**：內建 **86 分貝警報器**（可聞距離 180m）+ 指南針 app「回溯」功能（自動記錄路徑可返程） [^1]
- **A**：Ultra 獨有的戶外安全配置；警報器可引來救援者
- **B（獨自登山者）**：迷路時按警報器，同時「回溯」引導回到已知航點
- **B（夜跑者）**：遇到跟蹤者按警報器，引起周遭注意
- **B（帆船愛好者）**：人員落水可啟動警報器 + GPS 位置分享

## 常見客戶問題與回應

**Q1：Ultra 3 跟 Ultra 2 差在哪？**
A：主要升級 —（1）錶面角度從外視時更亮；（2）電池提升至 **42h**（Ultra 2 為 36h）；（3）雙頻 GPS 精準度超越市面雙頻運動錶；（4）watchOS 26 + 新功能。若你已有 Ultra 2，除非電池耗損才需換。

**Q2：Ultra 3 跟 Series 11 差在哪？要不要多花錢？**
A：值得升級的條件 —（A）**潛水**（Ultra 3 支援 40m，Series 11 僅泳池級 6m）；（B）**多日不充電**（42h vs 24h）；（C）**手套操作**（Action Button vs 觸控）；（D）**極端環境**（藍寶石玻璃 + IP6X + 100m 防水）。若以上都不需要 → 選 Series 11 更划算。

**Q3：女性戴 49mm 會不會太大？**
A：Ultra 3 只有 49mm 一個尺寸。建議直營店試戴 5 分鐘後再決定。許多女性戶外工作者 / 潛水員覺得 49mm 大小正好。若覺得過大 → 建議選 Series 11 46mm 鈦金屬款。

**Q4：可以取代專業潛水電腦嗎？**
A：休閒潛水（<40m、無減壓）完全可以；但**技術潛水**（混合氣、洞穴、沉船穿越）請仍使用專業潛水電腦。Oceanic+ 不取代 OW / AOW 認證訓練。

**Q5：Action Button 按錯怎麼辦？**
A：需要**連續兩秒長按**才觸發，避免誤觸。啟動後螢幕也會先確認，可取消。

## 常見反對意見處理

- **客戶說**：「我不潛水、不登山，用 Series 11 就好吧？」
  **你可以回應**：完全同意 — Ultra 3 是為極端場景設計的。除非你未來會走向馬拉松、三鐵、水肺潛水、高山縱走，否則 Series 11 CP 值更高。

- **客戶說**：「Garmin Fenix / Epix 能做的更多」
  **你可以回應**：Garmin 在純戶外訓練資料的深度確實更專業（肌肉負荷、壓力恢復、海拔適應），但 Ultra 3 的優勢是 **Apple 生態整合**（iMessage、Apple Pay、AirPods、iPhone 同步）+ **醫療級健康監測**（ECG、血氧、睡眠呼吸中止）。運動優先 → Garmin；運動 + 日常整合 → Ultra 3。

- **客戶說**：「這麼大一隻手錶上班戴會不會太誇張？」
  **你可以回應**：原色鈦金屬款其實相當低調；搭配皮革／彈力織紋錶帶更顯商務感。許多執行長日常也戴 Ultra — 是「一支錶搞定商務 + 戶外」的解法。
