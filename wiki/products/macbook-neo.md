## 五大賣點（Selling Points）

> 📌 依 `AGENTS.md` §4.5 FAB+P 原則撰寫。Benefits 段落為 persona-adaptive，腳本生成器依所選客群挑選對應句。
> 📌 定位說明：本機型為入門定位；A 層以「同價位筆電」為對照基準，而非 Apple 旗艦 MacBook Air/Pro。

### 1. 漂亮價格的 Mac 入門門票

- **F**：全新入門定位 MacBook，提供完整 macOS 與 Apple Intelligence 體驗 [^1]
- **A**：同價位 Windows 筆電無法提供 Mac 生態系 + 多年免費系統更新
- **B（學生）**：大學四年夠用；畢業時二手殘值高
- **B（家庭）**：孩子第一台電腦；爸媽第一次用 Mac 也能上手
- **B（商務）**：輕度差旅隨身機；作為公司配發 Mac 的補位機

### 2. A18 Pro 晶片 — AI 工具輕鬆跑

- **F**：Apple A18 Pro 晶片，支援完整 Apple Intelligence [^1]
- **A**：同價位 Windows 筆電需額外訂閱雲端 AI；端側運算保護個資
- **B（學生）**：ChatGPT、Canva、Microsoft 365 Copilot 全順跑
- **B（家庭）**：照片「清除」路人、書寫工具幫孩子改作文
- **B（商務）**：Zoom 視訊、PDF 摘要、英文郵件改寫一次到位

### 3. 四款鮮亮配色 + 同色鍵盤

- **F**：銀色、胭粉色、青橘黃色、靛青色；搭配同色系鍵盤；機身採 90% 再生鋁金屬 [^1]
- **A**：歷來最鮮亮多彩的 MacBook 系列；此價位段其他筆電無同色鍵盤設計
- **B（學生）**：宿舍書桌上與眾不同，朋友一看就知道是誰的
- **B（家庭）**：每個家人選自己顏色，視覺上分得清
- **B（商務）**：銀色款中性專業，場合皆適用

### 4. 13 吋 Liquid Retina 顯示器 + 空間音訊

- **F**：13" Liquid Retina（10 億色、500 尼特亮度、360 萬像素）+ 側射式雙揚聲器支援空間音訊 [^1]
- **A**：同價位 Windows 筆電多為 TN 或低色域 IPS；音訊亦無空間音訊支援
- **B（學生）**：線上課堂字體銳利；追劇音效環繞像戲院
- **B（家庭）**：全家追 Apple TV+ 或 Netflix 畫質音效雙滿分
- **B（商務）**：差旅飯店看劇放鬆；出差視訊會議聲音清晰

### 5. 與 iPhone 神奇聯手 + 內建安全

- **F**：接力、通用剪貼板、iPhone 鏡像輸出、AirDrop、即時熱點；內建病毒防護 + 免額外付費 macOS 更新 [^1]
- **A**：同價位 Windows 筆電無法與 iPhone 生態系連動；病毒防護需額外訂閱
- **B（學生）**：iPhone 寫一半的報告，Mac 打開接著寫
- **B（家庭）**：iPhone 拍的照片 AirDrop 到 Mac 整理；即時熱點家裡 Wi-Fi 壞了仍能上網
- **B（商務）**：iPhone 來電可在 Mac 接聽；通用剪貼板跨裝置拷貝


## 三大獨家 Demo（Signature Demos）

> 📌 **定位與展示原則**：MacBook Neo 為入門定位機型，規格上沒有同品牌旗艦獨家功能；以下 signature demos 依 `AGENTS.md` §4.1 選「此價位段最突出」功能（非行業獨家），並依 §4.2 採「預載素材 + 邊做邊說 + 口白 ≤15 字」節奏。

### Demo 1. iPhone 鏡像輸出 + 接力 — 從預錄短片起手，再現場做一次

**為何獨特**：在同價位帶的筆電中，只有 Mac 能把 iPhone 螢幕直接鏡射到電腦操作、用通用剪貼板無縫搬資料 — 這是 Windows 筆電無法複製的體驗。

**前置準備**：
- Demo 機已登入店員 Apple ID
- 桌面**預開「iPhone 鏡像輸出」App** 或有一段 15 秒示範短片（備援）
- 店員 iPhone 在身上，已登入同一 Apple ID，藍牙 + 同 Wi-Fi
- Keynote 預開一份空白簡報，游標待命

**分步腳本**（邊做邊說）：
1. 👉 打開桌面預開的「iPhone 鏡像輸出」視窗
   🗣️ 「您看，我的 iPhone。」
   👀 客戶看到 iPhone 畫面出現在 Mac 上
2. 👉 用 Mac 觸控板滑動，打開 Instagram 或訊息
   🗣️ 「手機在口袋裡，我沒碰。」
   👀 客戶驚訝 Mac 游標竟能操作 iPhone
3. 👉 在 iPhone 畫面複製一張圖片
   🗣️ 「手機上複製。」
   👀 出現「已複製」提示
4. 👉 切到 Mac 的 Keynote → Cmd+V
   🗣️ 「電腦上貼。」
   👀 圖片瞬間出現在 Keynote
5. 👉 打開 iPhone「郵件」寫一半 → Mac Dock 浮現 iPhone icon → 點擊繼續
   🗣️ 「路上寫，回家接著打。」
   👀 內容無縫延續

**客戶親手做**：請客戶用自己的 iPhone 試一次「複製訊息 → 貼到 Mac」— 離店前親手驗證「回家我也能這樣用」。

**常見卡點與應變**：
- 若 iPhone 鏡像首次需驗證 → 平時就把展示機設好信任關係；若無法 → 切換至預錄 15 秒示範短片播放
- 若客戶沒帶 iPhone → 用店員自己的 iPhone 示範，並強調「您家裡的 iPhone 回去就能這樣用」
- 若 Wi-Fi 或藍牙異常 → 跳過鏡像、改做接力 + 通用剪貼板（連線需求較低）

**收尾金句**：「一台 Mac，是 iPhone 最佳搭檔。」

### Demo 2. 四色同色鍵盤 + 再生鋁金屬 — 走到展示區，現場摸

**為何獨特**：MacBook Neo 是歷來最鮮亮多彩的 MacBook 系列（四色：銀／胭粉／青橘黃／靛青），同色系鍵盤是此價位帶其他筆電沒有的整體設計；再生鋁金屬重量占比 60% 為 Apple 產品歷來最高。

**前置準備**：
- 四款顏色展示機一字排開於展示檯
- 保留一台**青橘黃或靛青色**（最有視覺衝擊的顏色）開機待機、可讓客戶自由翻開操作
- 店員事先擦過機身，確保無指紋

**分步腳本**（邊做邊說）：
1. 👉 帶客戶走到四色陳列區
   🗣️ 「先看顏色。」
   👀 客戶一眼感受四色陣列
2. 👉 請客戶拍一張俯瞰照
   🗣️ 「拍起來，回家看。」
   👀 客戶主動參與、建立擁有感
3. 👉 請客戶翻開他最喜歡的那一台
   🗣️ 「鍵盤也是同色。」
   👀 客戶注意到機身與鍵盤呼應
4. 👉 請客戶手放掌托、指放鍵盤
   🗣️ 「摸摸看 — 再生鋁金屬。」
   👀 觸感冰涼、紮實
5. 👉 邀請客戶打幾個字
   🗣️ 「打幾個字試試。」
   👀 客戶體驗巧控鍵盤回饋
6. 👉 翻開第二台不同顏色
   🗣️ 「再摸一台，比比看。」
   👀 客戶完成「挑選」的心理動作

**客戶親手做**：請客戶同時翻開兩台不同顏色，實際比較掌托觸感與鍵盤光澤，請他說出最喜歡哪一款。

**常見卡點與應變**：
- 若四色展示機不齊 → 優先展示 2-3 色，並告知「其他顏色可於 apple.com 或 Apple Store App 選購」
- 若客戶覺得顏色太搶眼 → 直接引導到銀色，強調中性適合商務
- 若客戶對顏色無反應 → 切換為環保永續訴求，強調 60% 再生物料是 Apple 歷來最高

**收尾金句**：「陪您 8 小時，挑個會微笑的顏色。」

### Demo 3. Apple Intelligence「清除」— 從預載風景照開始，邊滑邊改

**為何獨特**：A18 Pro 晶片讓 MacBook Neo 在入門價位也能完整執行 Apple Intelligence；**「清除」**功能於 Mac 端側運算、不上傳雲端 — 這是同價位 Windows 筆電需額外訂閱雲端 AI 服務才能做到的。

**前置準備**：
- Demo 機已啟用 Apple Intelligence（繁中）、iOS 26/macOS 版本就緒
- 相簿**預載一張有明顯干擾物的風景照**（例：海灘照有路人、山景照有電線）
- 桌布可切換為簡潔底色，避免分心

**分步腳本**（邊做邊說）：
1. 👉 打開「照片」App → 選預載風景照
   🗣️ 「您看這張海邊。」
   👀 客戶注意到後面有個路人
2. 👉 「編輯」→「清除」→ 用游標圈選路人
   🗣️ 「圈起來。」
   👀 選取動作直覺
3. 👉 放開游標 → AI 自動填補背景
   🗣️ 「放開，乾淨了。」
   👀 干擾物消失，背景自然
4. 👉 左右滑比對原圖
   🗣️ 「一前一後。」
   👀 客戶看到明顯改善
5. 👉 點左上角資訊
   🗣️ 「運算沒離開這台電腦。」
   👀 客戶同時感受到「效果」與「隱私」

**客戶親手做**：請客戶用 AirDrop 傳一張自己的照片（旅遊／寵物／美食皆可），現場幫他清除一個干擾物 — 這是最強的情感記憶點。

**常見卡點與應變**：
- 若預載照片被刪 → 當場打開「相機」拍一張「桌面上放把鑰匙」的照片，現場清除鑰匙作示範
- 若 AI 運算結果不完美（出現殘影）→ 示範「再圈一次」修正，誠實說「AI 不是萬能但夠用」
- 若 Apple Intelligence 正在下載中 → 改用「書寫工具」改寫語氣 demo（備忘錄輸入一段話）

**收尾金句**：「入門價格，AI 不打折。」

## 常見客戶問題與回應

**Q1：MacBook Neo 跟 MacBook Air 差在哪？**
A：三個主要差異 — Neo 搭載 A18 Pro（Air 是 M5 晶片）、電池 16 小時（Air 18 小時）、Neo 有 2 個連接埠（Air 有 4 個含 Thunderbolt 與 MagSafe）。Neo 定位入門價格、漂亮配色；Air 定位更輕薄、更高效能。詳見 [[comparisons/macbook-neo-vs-air]]。

**Q2：A18 Pro 跟 M 系列晶片差多少？**
A：A18 Pro 原本是 iPhone 等級的高階晶片，日常 app、輕度剪片、AI 工具都夠用。但若您常做重度影像剪輯、3D 工作、開發程式，建議選 M 系列的 MacBook Air / Pro。

**Q3：可以取代 PC 嗎？**
A：對大多數文書、瀏覽網頁、影音娛樂、線上課程完全沒問題。若有 Windows 專用軟體需求（特定遊戲、某些公司內部系統），需先確認是否有 Mac 版或替代方案。

**Q4：跟 iPad 比該選哪個？**
A：要用鍵盤打報告、管理檔案、跑完整桌面版 app → 選 MacBook Neo。要觸控手寫、繪圖、輕薄閱讀 → 選 iPad Air。兩者也能互補使用。

**Q5：這台會過時嗎？可以用幾年？**
A：Apple 免額外付費提供多年 macOS 更新，配合定期軟體更新與病毒防護，一般使用 4-5 年以上無問題。

**Q6：有哪些付款與優惠方案？**
A：可享最長 1 年零利率分期、Apple Trade In 舊機換購折抵、學生與教職員教育優惠、Apple Store App 下單個人化設定服務。

## 常見反對意見處理

- **客戶說**：「只用 A18 Pro 會不會效能不夠？」
  **你可以回應**：理解您的擔心。A18 Pro 設計時就預期要跑 Apple Intelligence 與 ChatGPT 等 AI 工具，日常使用非常充裕。若您主要用途是文書、上網、追劇、視訊、輕度剪片，這顆晶片綽綽有餘；若您需要專業影像後製或開發工作，我會推薦 MacBook Air 的 M5 會更適合。

- **客戶說**：「連接埠只有 2 個不夠用」
  **你可以回應**：這個考量很實際。若您常外接多個螢幕、外接硬碟、有線鍵鼠等，建議升級 MacBook Air（4 個連接埠含 Thunderbolt）。若您是輕度使用偶爾接隨身碟，2 個 USB-C 是夠的，也可以搭配 USB-C 集線器擴充。

- **客戶說**：「PC 比較便宜啊」
  **你可以回應**：同價位段確實有更便宜的 PC。Mac 的優勢在長期體驗 — 免費系統更新多年、內建病毒防護不用額外付費、與 iPhone 無縫接軌、二手殘值高。如果您家中已有 iPhone，Mac 會特別有感。

- **客戶說**：「顏色會不會太鮮豔不適合辦公場合？」
  **你可以回應**：完全理解。我們提供四款顏色，其中銀色相當中性，適合正式商務場合。胭粉、青橘黃、靛青則適合想要個人風格的使用者。
