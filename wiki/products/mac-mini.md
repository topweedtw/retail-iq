# Mac mini (M4 / M4 Pro)

## 一句話定位

**最迷你、最超值的 Mac，實力超強大**。**12.7 × 12.7 公分**見方、**0.67 公斤**極精密機身，搭配 M4 或 M4 Pro 晶片、最多 8 個連接埠（含前置 2×USB-C）、硬體加速光線追蹤，較同價位 PC 桌機**最高快 6 倍**。[^1]

## 起售價

- **Mac mini（M4，10C CPU / 10C GPU / 16GB RAM / 512GB SSD）**：**NT$ 26,900 起** [^1]
- **Mac mini（M4 Pro，12C CPU / 16C GPU / 24GB RAM / 512GB SSD）**：**NT$ 46,900 起** [^1]

> 備註：2026 年起起售價隨 256GB SSD 基礎款停產而調整，最低配置從 512GB SSD 起跳 [^2]。

## 目標客群

- **[[customer-personas/home-office]]** — 家用 / 桌面雙螢幕辦公
- **[[customer-personas/developer]]** — 軟體工程師、自架伺服器
- **[[customer-personas/creator]]** — 剪片、修圖、3D 建模（含 M4 Pro）
- **[[customer-personas/student]]** — 學生書桌的最低成本 Mac
- **[[customer-personas/small-business]]** — 店家 POS、小型工作室
- **[[customer-personas/mac-upgrader]]** — 用舊 Mac mini 或 PC 想升級

## 五大賣點（Selling Points）

> 📌 依 `AGENTS.md` §4.5 FAB+P 原則撰寫。Benefits 段落為 persona-adaptive。

### 1. 歷來最小 Mac 桌機 — 12.7×12.7 公分，0.67 公斤

- **F**：**12.7 × 12.7 公分**見方、僅 **0.67 公斤**；相比 2023 版（19.7 cm）體積只有 1/20 [^1]
- **A**：一本小說的體積就裝下完整 Mac 桌機；可藏於螢幕後、架高、甚至隨身攜帶到辦公室
- **B（家用辦公）**：書桌前不再有主機佔空間，螢幕下或牆掛都 OK
- **B（小店家）**：收銀台下塞一台就成 POS，外出展覽放進背包也行
- **B（數位遊牧）**：出差時背一顆電源 + Mac mini，到飯店接螢幕即開工

### 2. M4 / M4 Pro — 較同價位 PC 桌機最快 6 倍

- **F**：配備全球最快 CPU 核心、硬體加速光線追蹤、16 核心神經網路引擎 [^1]
- **A**：較相同價位暢銷 PC 桌機最快 **6 倍**（Apple 實測）；M4 Pro 光線追蹤速度是 M4 的 2×
- **B（程式工程師）**：Xcode 預測程式碼 + 多模擬器同時建置，完全不頓挫
- **B（影音創作者）**：Final Cut Pro 4K ProRes 剪輯、DaVinci Resolve 算圖飛快
- **B（遊戲玩家）**：《Control Ultimate Edition》《Myst》光線追蹤反射一路開到最高
- **B（研究人員）**：MATLAB 大資料集、Blender 3D 建模、Autodesk AutoCAD，小機身也扛得住

### 3. 最多 8 個連接埠 + 前置 USB-C — 插拔無需轉身

- **F**：**正面** 2× USB-C + 3.5 mm 耳機孔；**背面** 3× Thunderbolt（M4 Pro 是 Thunderbolt 5，達 120 Gb/s）+ HDMI + Gigabit 乙太網路 [^1]
- **A**：前置 USB-C 解決「每次插 USB 都要伸手到背後」的日常麻煩；Thunderbolt 5 是業界最快 I/O
- **B（攝影師）**：SD 讀卡機 + 硬碟從前方插，拍完立即匯入
- **B（影片剪輯）**：RAID 外接陣列透過 Thunderbolt 5 直通 120 Gb/s，ProRes 即時預覽
- **B（多螢幕工作者）**：M4 Pro 撐 3 台 6K 螢幕 + 鍵鼠 + 耳機 DAC，背面插滿還留前置給隨插

### 4. 整機 AI 加速 + 為 Apple Intelligence 打造

- **F**：16 核心神經網路引擎 + macOS Tahoe 原生整合 Apple Intelligence（書寫工具 / Genmoji / 影像樂園 / 清除 / 自然語言搜尋）[^1]
- **A**：裝置端 AI 處理（資料不離開 Mac）+「私密雲端運算」（需更大模型時仍保隱私）
- **B（內容創作者）**：用 ChatGPT + Canva + Final Cut 同時開，效能不卡
- **B（寫作者）**：書寫工具一鍵潤稿、換語氣、做摘要，節省 1/3 時間
- **B（攝影師）**：「清除」功能一按去掉路人 / 電線干擾
- **B（重視隱私者）**：個資始終在本機，不會上傳蘋果伺服器

### 5. 與 iPhone 無縫整合 — 接電話、鏡像、AirDrop

- **F**：iPhone 鏡像輸出（從 Mac 操作 iPhone）、通用剪貼板、AirDrop、來電／簡訊在 Mac 接聽收發 [^1]
- **A**：Mac mini 放桌上就能接 iPhone 電話、收訊息、拷貝貼上跨裝置
- **B（遠距工作者）**：iPhone 放包包，Mac mini 接電話也沒問題
- **B（多裝置使用者）**：在 iPhone 拍的照片，直接拖進 Mac 的 Photoshop
- **B（出差族）**：旅館無 Wi-Fi？iPhone 即時熱點給 Mac mini 用


## 三大獨家 Demo（Signature Demos）

> 📌 展示原則：Mac mini 的核心驚奇是「**小身體大實力**」。展示需要讓客戶**親手拿起機身**並操作。

### Demo 1. 拿起 0.67 kg 機身 + 全速啟動 Final Cut

**為何獨特**：多數客戶以為 Mac mini 很大（記得 2023 版）。讓他們親手拿起新 Mac mini 會是震撼體驗。

**前置準備**：
- Demo 機是 M4 Pro 款（讓效能更驚人）
- Final Cut Pro 預載一個 4K 多軌剪輯專案
- 螢幕一側放 2023 舊版 Mac mini（視覺對比用）
- 體重秤放桌上（「看，不到 1 公斤」）

**分步腳本**：
1. 👉 把 Mac mini 拿起來交給客戶
   🗣️ 「先拿拿看。」
   👀 客戶驚訝它這麼輕
2. 👉 指著旁邊的 2023 舊版
   🗣️ 「體積只有以前的 1/20。」
3. 👉 放回原位 → 打開 Final Cut Pro 預載專案
   🗣️ 「播 4K 多軌。」
   👀 預覽順暢、沒有 buffering
4. 👉 加 3 個特效同時播放
   🗣️ 「加效果，還是順。」
   👀 CPU / GPU 表在 Activity Monitor 上不爆
5. 👉 匯出 1 分鐘 4K → 計時
   🗣️ 「這種尺寸的電腦 — 跑這種任務 — 以前想都不敢想。」

**客戶親手做**：讓客戶把 Mac mini 拿在手上，感受重量 + 尺寸。

**常見卡點**：
- 客戶問「這個可以塞進車用儀表板嗎」→ 強調「技術上可以，但需足夠散熱 + 電源」
- 非影音用戶 → 改示範多瀏覽器分頁 + Excel + Zoom 同時開

**收尾金句**：「桌機等級的效能，放進書桌抽屜。」

### Demo 2. 前置 USB-C — 接上 SD 卡讀卡機直接匯入

**為何獨特**：前置 USB-C 是小細節但極實用 — 幾乎所有其他桌機都要伸手到背後。

**前置準備**：
- 準備一個 USB-C SD 卡讀卡機 + SD 卡（含測試照片）
- 照片 App 已就緒
- Demo 機正面面向客戶

**分步腳本**：
1. 👉 指出正面兩個 USB-C 孔
   🗣️ 「看前面 — 兩個 USB-C。」
2. 👉 把讀卡機插進前方 USB-C
   🗣️ 「以前要繞到背後 — 現在伸手就到。」
3. 👉 SD 卡插入 → 照片自動匯入
   🗣️ 「拍完回來，一插就匯入。」
   👀 進度條快速完成
4. 👉 強調後方仍有 3× Thunderbolt + HDMI + 乙太網路
   🗣️ 「固定裝置插後面、隨用插前面。」
5. 👉 拿出 Thunderbolt 外接硬碟 → 插到後方 TB 5 孔
   🗣️ 「TB 5 — 120 Gbps，影片即時開啟。」

**客戶親手做**：讓客戶自己把 SD 卡拔出重插一次，感受位置方便性。

**收尾金句**：「為日常而設計，不是為照片規格而設計。」

### Demo 3. M4 Pro 光線追蹤 — 遊戲 / 3D 算圖展示

**為何獨特**：大多數「小桌機」在提到「硬體光線追蹤」時會笑場 — M4 Pro Mac mini 不會。

**前置準備**：
- Demo 機為 M4 Pro 配置
- 預裝《Control Ultimate Edition》或《Myst》支援硬體光線追蹤
- Blender 或 Cinema 4D 預載 3D 場景
- 外接 4K 顯示器（展示渲染細節）

**分步腳本**：
1. 👉 開《Control》 → 進入有反射場景的關卡
   🗣️ 「光線追蹤開。」
   👀 地板反射、玻璃折射、光影逼真
2. 👉 切設定關閉光線追蹤
   🗣️ 「關了再看一次。」
   👀 差異非常明顯
3. 👉 再切回開啟 → 幀率不掉
   🗣️ 「開啟 — 依然順。」
4. 👉 切到 Blender 或 Cinema 4D
   🗣️ 「換 3D 工作來看。」
5. 👉 跑一個 Redshift / Octane 算圖（短片段）
   🗣️ 「一分鐘的場景算圖 — 計時開始。」
   👀 M4 Pro 幾秒內完成

**客戶親手做**：請客戶拿手把試玩《Control》2 分鐘。

**常見卡點**：
- 客戶是 PC 用戶 → 可比較同價位 PC 需要多大的主機才能跑到一樣效果
- Mac 遊戲庫不如 PC → 承認此點，但強調「大作持續登陸 Mac（Control / Cyberpunk / Death Stranding 等）」

**收尾金句**：「這麼小的盒子 — 能做這樣的事。」

5. **macOS Tahoe**：Liquid Glass 新設計、Spotlight、app 切換
6. **Thunderbolt 5 速度**：M4 Pro 機型插 TB 5 外接硬碟 → 1GB 檔傳輸 < 2 秒

## 常見客戶問題與回應

**Q1：M4 還是 M4 Pro？**
A：三個判斷點 —（1）**剪 4K+ / 3D 建模 / 多 6K 螢幕** → M4 Pro；（2）**一般文書 / 輕度創作 / 2 螢幕以下** → M4；（3）**預算緊 + 未來可能升級**：買 M4 + 最大 24GB RAM，過三年再升級整台，仍比一開始買 M4 Pro 便宜。

**Q2：Mac mini 沒有螢幕 / 鍵盤 / 滑鼠，划算嗎？**
A：這正是它的彈性 — 可以用家裡現有螢幕 / 電視。如果要搭 Studio Display + 巧控鍵盤 + 巧控滑鼠全套 Apple 周邊，價格會接近 iMac。**若你看重空間彈性、想自選螢幕** → Mac mini；**若你要一體成型、開箱即用** → iMac。

**Q3：Mac mini vs MacBook Neo / Air？**
A：**需要攜帶** → MacBook；**總是在桌上 + 省錢 + 彈性** → Mac mini。Mac mini 的效能 / 價格比筆電強，但多了螢幕 / 鍵盤的配置成本。

**Q4：記憶體 / 儲存可以事後升級嗎？**
A：**不行**。Apple Silicon 的統一記憶體是封裝在 SoC 上，SSD 也是焊接。購買時要一次選對。建議記憶體至少 16GB，儲存 512GB 起（內部儲存用完就只能靠外接）。

**Q5：Thunderbolt 4 跟 Thunderbolt 5 差多少？**
A：TB 4 = 40 Gbps；TB 5 = **80 Gbps（M4 Pro 的 Mac mini 是 120 Gbps 特規）**。對於 4K+ 多軌影片、RAID 儲存、多顯示器的用戶 → TB 5 的差異非常有感。

## 常見反對意見處理

- **客戶說**：「PC 桌機更便宜，規格表看起來差不多」
  **你可以回應**：Apple 自己的測試 — M4 Mac mini 較同價位 PC 桌機快 **最多 6 倍**（特別是影音、AI、3D 任務）。而且 Mac mini 用料、靜音、能耗、macOS 整合性都比同價位 PC 好。規格表的「Hz / 核心數」不完全反映實用效能。

- **客戶說**：「我已經有 iMac / MacBook Pro，不需要再一台」
  **你可以回應**：Mac mini 通常是**當第二台 Mac 的好選擇** —（1）當家中 NAS / 自動化主機（全天開機）；（2）當影音工作站（iMac 螢幕受限）；（3）當小朋友 / 家人獨立機（各用各的，但共享 iCloud）。

- **客戶說**：「沒螢幕要另外買，不划算」
  **你可以回應**：你家電視可當螢幕（HDMI 直接接）；或舊 PC 螢幕拿來用（HDMI / DisplayPort 都可）。若本來就要買新螢幕，Studio Display 是最匹配搭檔；不然便宜的 4K 螢幕 + Mac mini 反而能花同樣錢買到**更高階的 Mac 晶片**。

- **客戶說**：「這麼小 — 會不會很熱 / 噪音大？」
  **你可以回應**：Apple 特別為這尺寸重新設計散熱架構（氣流引導到底座）。加上 Apple Silicon 本身能效遠優於 Intel，實測一般使用幾乎聽不到風扇。影音重負載才會啟動風扇，但音量仍遠低於多數 PC 桌機。
