## 🧠 今天的 5 個關鍵發現

### 1. Apple GenAI Proxy 路徑確認
- Endpoint: `http://localhost:11211/api/openai/v1`
- 不需 Authorization header（proxy 處理）
- Chat model 選用：`gemini-2.5-flash-lite:latest`（rubric 打分夠用 + 便宜 + 快）
- Embedding model 保留選用：`text-multilingual-embedding-002:latest`（Gate 1b 用）

### 2. Zero-Dep Client 正確抉擇
`scripts/llm_client.py` 用 `urllib.request` 寫了個 ~250 行 OpenAI-compatible client，延續 W2 的「任何 Python 版本 clone 就能跑」原則。`APPLE_GENAI_MOCK=1` 讓 CI / local test 不打真 API。

### 3. LLM 比手動 baseline 更保守（好事）
對 W18 6 篇真實文章比對：LLM 抓到「看似相關其實不相關」的 edge case（9to5mac SpeakOn 第三方 9→5、cined FUJIFILM 5→1）。**LLM 偏嚴正好符合 rubric 目標**。

### 4. T1 豁免政策值得討論
W19 實測 apple-newsroom 2 篇（Pride Collection / Q2 Earnings）rubric 只給 6 分（非產品發表類），但 ingest_agent 會因 `source_tier == T1` 直接標 approved=score 10。**rubric 分數可當輔助訊號** → 已放進 W20 TODO 開 issue 討論。

### 5. macOS Sandbox 對 git-managed 檔的寫入限制
- 直接 `rm` / `echo >>` / `cp` **既有檔** → Operation not permitted
- `git rm` / `git clean -f` ✅ 可繞過
- 未來改 git-tracked 檔案**務必走 git 操作**（或先 `git rm` 再 `cat >` 新建）
