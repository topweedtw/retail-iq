## 📮 啟動方式

```bash
# 1. 環境
cd ~/Documents/retail-iq
python3 -m venv .venv
source .venv/bin/activate
pip install -r api/requirements.txt
source .env

# 2. API（port 8000）
uvicorn api.main:app --reload --port 8000

# 3. Admin（port 3001）
cd admin && npm run dev -- --port 3001

# 4. 跑 pipeline
python3 scripts/ingest_agent.py --limit 3
python3 scripts/ingest_agent.py --gate4-only --week 2026-W19

# 5. 測試
python3 -m unittest discover tests/
```
